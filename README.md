# ISHU-SERVER API

**Installation**
-----

- Setup Node.js
        
    https://nodejs.org/en/
        
- Command Line
    
    Install node packages: `npm install`
    
    Build project: `npm build`
    
    Serve: `npm run start`
    
**Overview**
-----
- API Base URL: http://localhost:5000/v1

- API Doc: http://localhost:5000/api-docs
